# **Modele int_calendar**

**Description** :

Modele de la couche intermediate.Ce modele permet de créer un calendrier dénormalisée . Les attributs descriptifs pourront ensuite 
être utilisé à differents niveaux de granularité notemment dans la couche mart.
Le modele est généré a partir de variables définies dans dbt_project.yml pour l'etendue de dates
      
**Granularité** :
La granularité la plus fine ici est le jour (pas de notion d horaire)
