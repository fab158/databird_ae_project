# **Modele int_products_joined**

**Description** :

Table de la couche intermediate. Cette table sert à denormaliser les données du produit.
Il s'agit donc ici de mettre a plat les informations liées au concept de produit (brand, categorie..) etc
Cette agregation pourra etre ensuite etre reutilisée par les couches superieures

**Granularité** :

La granularité la plus fine ici est le produit
